# Linear Regression using NumPy, Pandas & Matplotlib

This project implements **Linear Regression from scratch** using **Python** with the help of **NumPy**, **Pandas**, and **Matplotlib**.  
The objective of this project is to understand how linear regression works mathematically and how data can be analyzed and visualized effectively.

## Author
**Nittesh Pandey**

## Technologies Used
- Python
- NumPy
- Pandas
- Matplotlib

## Project Description
Linear Regression is a fundamental algorithm in Machine Learning and Data Analytics.  
In this project, NumPy is used for numerical computations, Pandas for data handling, and Matplotlib for visualizing data points and the regression line.

## Features
- Linear Regression implementation from scratch
- Data handling and analysis using Pandas
- Numerical computation using NumPy
- Data visualization using Matplotlib
- Clean and beginner-friendly code
